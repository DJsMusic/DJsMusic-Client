Technology

DJs Music has been rebuilt using the latest web technologies and standards. The client (What you are seeing right now) and Server have been completely separated, for a better user experience and ease of development.

Below are the technologies, libraries and scripts used (Obvious dependencies like jQuery have been omitted)

Client

  • Backbone.js

    Handles all the application logic.

  • Require.js

    Loading the different scripts.

  • Sound Manager 2

    Provides an audio API for the custom player (See below).

  • Bootstrap

    Design components for the application

  • Bootstrap Slider

    Player seek function.

  • Nanobar

    Loading indicator (Similar to the one YouTube uses)

Github

Server

  • Slim Framework

    Backend RESTful API written in PHP

  • MySQL

    Database engine

  • PHPUnit

    Testing framework

Audio Player

The audio player has been designed and coded from scratch becoming a standalone "Backbone player": it accepts both Song models and collections, which allows for an extremely simple use. Whenever a Song model is used, a Play button can be inserted with no extra coding.

It uses Sound Manager 2 to play the sounds, although at the moment there is not much more integration with the library. One of the key aspects of this library is the fact that it works on most devices and browsers out of the box.
